# Elaborato2021
is a Python software for send and receive email with cypher attachments.

## Installation

For the correct use of the software you need to install:

- GPG software

```bash
sudo apt install gpg
```
- gnupg

```bash
pip install python-gnupg
```
- imap-tools

```bash
pip install imap-tools
```
